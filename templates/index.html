This tool is built with Multinomial Naive Bayes sentiment classifier. This will collect local tweets related to the input using hashtag. The collected tweets will be processed and will classify as negative or positive sentiments. A time-series line graph and word cloud will be seen in the Visualization Result of Sentiment Analysis together with Model Evaluation Result and Sentiment Predictions.
